# UltraRAG 3.0：让每一行推理逻辑都看得见

我们很高兴地宣布 **UltraRAG 3.0** 正式发布！

## 为什么发布 3.0？

在 RAG（检索增强生成）应用的开发过程中，我们发现开发者常常面临“盲盒”困境：
- 检索召回了什么？不知道。
- 模型为什么这么回答？不清楚。
- 逻辑链条在哪里断了？难以调试。

UltraRAG 3.0 旨在解决这些问题，提供全链路的白盒化推理体验。

<!-- truncate -->

## 核心特性

### 1. Pipeline Builder
所见即所得的逻辑编排。自动化处理繁琐界面封装，只需专注于逻辑编排，静态代码即刻变身可交互的演示系统。

### 2. White-box Reasoning
拒绝黑盒。实时呈现复杂长链条任务中的每一次循环、分支与决策细节。

### 3. AI Copilot
内嵌懂框架的 AI 助手，通过自然语言交互辅助生成 Pipeline 配置与优化 Prompt。

## 快速开始

```bash
pip install ultrarag
ultrarag init my-project
```
